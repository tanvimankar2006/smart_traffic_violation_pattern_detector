## ğŸ“Š How It Works

### **Global Filtering**
1. Select filters in sidebar (States, Violation Types, Weather)
2. **ALL** visualizations use the same filtered data
3. Charts update automatically when filters change
4. KPIs always match chart values

### **Navigation Flow**
1. User selects a feature via radio button in sidebar
2. That feature's content loads in the main area
3. Previous content disappears (clean, uncluttered)
4. Filters apply globally to all features

### **Plotting System**
8 reusable plotting functions centralize all styling:
- `plot_count()` - Categories as bars
- `plot_hist()` - Distributions
- `plot_line()` - Time trends
- `plot_bar()` - Vertical comparisons
- `plot_barh()` - Horizontal rankings
- `plot_scatter_with_ref()` - Relationships with annotations
- `plot_heatmap()` - Numeric correlations
- `plot_pie()` - Percentages (pie or donut)
